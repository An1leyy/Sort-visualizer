# Sort Visualizer

**Интерактивная визуализация алгоритмов сортировки с плавной анимацией**

</div>

---

## Описание

**Sort Visualizer** — это инструмент, который позволяет наглядно увидеть, как работают различные алгоритмы сортировки. Программа визуализирует процесс сортировки массива с помощью цветных столбцов, показывая каждое сравнение и обмен элементов в реальном времени.

---

## Особенности

### Алгоритмы сортировки
- **Пузырьковая сортировка** (Bubble Sort)
- **Сортировка вставками** (Insertion Sort)  
- **Сортировка выбором** (Selection Sort)

### Визуальные эффекты
- Столбцы с закругленными углами
- Подсветка сравниваемых элементов (оранжевый цвет)
- Плавное перемещение обмениваемых элементов
- Отображение числовых значений под столбцами

### Интерфейс
- Интерактивные кнопки с кастомными текстурами
- Горячие клавиши для быстрого управления
- Импорт массивов из файла
- Пошаговый и автоматический режимы
- Настраиваемая случайная генерация массива
- Плавная анимация перемещения столбцов

pip install -r requirements.txt

## Управление

### Клавиатура

| Клавиша | Действие |
|---------|----------|
| `B` | Запустить пузырьковую сортировку |
| `I` | Запустить сортировку вставками |
| `S` | Запустить сортировку выбором |
| `Пробел` | Шаг вперед (в пошаговом режиме) |
| `A` | Переключение автоматического режима |
| `R` | Сгенерировать новый случайный массив |
| `ESC` | Выход из программы |

### Кнопки мыши

| Кнопка | Действие |
|--------|----------|
| **Bubble** | Запуск пузырьковой сортировки |
| **Insertion** | Запуск сортировки вставками |
| **Selection** | Запуск сортировки выбором |
| **Import from file** | Импорт массива из файла |
| **Reset** | Сброс к исходному массиву |
| **Random** | Генерация случайного массива |
| **Play/Pause** | Автоматическое воспроизведение |
| **Step** | Пошаговый режим |

---

## Импорт массива из файла array.txt

Для импорта массива запишите массив в файл расширения .txt в формате "1 2 3 4 5 6 7".
Максимально допустимая длина массива — 50 элементов.

---

## Настройка генерации массива

### Конфигурационный файл

Программа поддерживает настройку параметров генерации случайного массива через JSON файл. Это позволяет изменять диапазон значений и размер массива без редактирования кода.

### Файл `array_config.json`

Создай файл `array_config.json` в корневой папке проекта со следующим содержимым:

```json
{
    "min_length": 10,
    "max_length": 50,
    "min_element": -999,
    "max_element": 1000
}
```
### Параметры конфигурации

| Параметр | Описание | Допустимые значения |
|--------|----------|----------|
| `min_length` | Минимальный размер массива | 1 - 50 |
| `max_length` | Максимальный размер массива | 1 - 50 |
| `min_element` | Минимальное значение элемента | любое целое число |
| `max_element` | Максимальное значение элемента | любое целое число |

> [!NOTE]
> Числа, выходящие за границы [-9999, 9999] будут представлены в научной форме.

### Условия работы

Если значения в файле некорректные или самого файла не существовует, программа будет использовать стандартные константные значения при генерации массива.
